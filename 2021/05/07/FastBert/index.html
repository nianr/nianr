FastBert -- An improvement for BERT

Posted on 2021-05-06 Edited on 2021-05-07

中文版

Abstract

  • Original paper
  • adaptive inference time
  • unique self-distillation mechanism at fine-tuning
  • speed up by 1 to 12 times than BERT

Summary

  • FastBert has two mechanisms:
  1. sample-wise adaptive mechanism –> Adaptive inference
  2. unique self-distillation process –> Knowledge distillation
  • This picture shows the framework of FastBert：
  • According to it, FastBert has two parts, one is Backbone another one is Branch.
  • Backbone contains 12 layers of transformers, similar to the stucture of BERT. As to Branch, it distillings the student model from different layers.

Backbone

  • All the embedding layers inside the 12 transformers are the same as BERT.
  • The different from BERT is that it has an additional Teacher-Classifer, the structure is as follows:
  • Transformer outputs 768-dims features. They are compressed to 128-dims by a full connected layer firstly, then though self-attention layer witch doesn’t change the dimension. And then they though the second full connected layer, compressed to N-dim, where N is the the number of classification categories. Finally we get the probability pt of each category by applying sofmax.

Branch

  • Branch contains L-2 Student-Classifer，which are distilled from different layer of transformers.
  • The structure of Student-Classifer is the same as Teacher-Classifer。

Pre-training and Fine-tune

  • Two steps for training a FastBert model:
  1. pre-train Backbone
    • In this step, the parameters in Teacher-Classifer are unchanged, so the pre-training process is exactly the same as that of BERT.
    • As a result, we can directly use the parameters of the Bert model.
  2. fine-tune Backbone
    • This step aims to train Teacher-Classifer, and downstream tasks need to be determined before training.
    • In this step, Student-Classifer is unchanged.
  3. fine-tune Branch
    • This step is to train Student-Classifer by using the result of Teacher-Classifer. The author defines the following loss function:

    • KL divergence is relative entropy, and the total loss function is the sum of the KL divergence between Teacher and Student. (ps: I don’t know the name of “KL divergence” clearly, so please check it further if you want to use it in a formal article. )
    • Label is not used in this step of training, so we can use unsupervised training Student-Classifer。

Adaptive inference

  • Adaptive inference in FastBert is “sample-wise”, which means that for each sample, the model will adaptively choose which Student-Classifer to use. For this reason, the author has made the following definition:
  • After each layer of transformer, the uncertainty of each categroy is computed. The above formula is actually information entropy.
  • Then the author made an assumption that the lower the uncertainty, the higher the accuracy of the model. So he set a threshold speed for the uncertainty, and those that exceed this threshold are passed to the next layer of transformer, others are classified by the current Student-Classifer.
