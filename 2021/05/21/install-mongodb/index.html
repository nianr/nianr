Install MongoDB on MacOS

Posted on 2021-05-20 In Database

Follow the tutorial provided by official documents, link here .

Installation

  1. Make sure you have installed Xcode command-line tools, if not, run the following command:
    xcode-select --install
  2. Install Homebrew using this instruction
  3. Tap Homebrew to download the Homebrew formula of MongoDB:
    brew tap mongodb/brew
  4. Instal a community version of MongoDB:
    brew install mongodb-community@4.4

Run MongoDB

  1. To start a MacOS service using brew, issue the following:
    brew services start mongodb-community@4.4

    ps: The brew service will start as soon as MacOS launched.

  2. To stop this service:
    brew services stop mongodb-community@4.4
  3. To run MongoDB as a background process, issue the following:
    mongod --config /usr/local/etc/mongod.conf --fork

    ps: This process won’t relaunch after you stop it or shut down your MacOS.

  4. To kill this process:
    ps aux | grep -v grep | grep mongod

Connect to MongoDB

  1. Type the following on a new terminal window:
    mongo
  2. To tun mongotop (a MongoDB tool), issue the following:
    mongotop
